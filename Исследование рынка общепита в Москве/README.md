**Исследования рынка общепита в Москве для принятия решения об открытии нового заведения** - Мною был исследован вопрос: будет ли успешным и популярным на долгое время кафе, в котором гостей обслуживают роботы-официанты. По результатам анализа подготовлена презентация для инвесторов с рекомендациями.
